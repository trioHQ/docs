Welcome to the Trio API documentation!

The best way to get started is by creating your Sandbox account. This guide walks you through the initial setup and prepares your environment for testing all Trio APIs.

👉 **Start here:** [/guides/sandbox_account](/guides/sandbox_account)

Once your Sandbox account is ready, you'll be able to:
- Generate your API credentials.
- Configure webhooks.
- Test all available banking APIs in a safe environment.
- Follow the integration guides throughout this documentation.

<Note>
  If you're new to Trio, we recommend completing the Sandbox Account guide before exploring the API Reference.
</Note>
